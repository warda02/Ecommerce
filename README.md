# 🛒 E-Commerce Card Shop

A responsive multi-page e-commerce website built with vanilla HTML, CSS, and JavaScript.

## 🔗 Live Demo
[View Project](https://warda02.github.io/Ecommerce)

## ✨ Features
- 🏠 Home page with product listings
- 🛍️ Shop page with product cards
- 🔐 Login & Sign-up pages
- 🧾 Checkout page
- 📱 Fully responsive design

## 📁 Project Structure
```
Ecommerce/
├── index.html       # Home page
├── shop.html        # Product listing
├── checkout.html    # Checkout page
├── loginin.html     # Login page
├── signin.html      # Sign up page
├── assets/          # CSS & JS files
└── image/           # Product images
```

## 🚀 Getting Started
```bash
git clone https://github.com/warda02/Ecommerce.git

# Open index.html in your browser
```
